Plugin pro [WordPress][1], který umožňuje generovat smuteční oznámení.

== Description ==

Hlavní vlastnosti:

* vytvořená oznámení jsou uložena pro opětovné použití jako _custom post type_ a mohou být díky tomu i náležitě používána
* _shortcode_ se samotným designérem smutečních oznámení
* v nastavení pluginu umožňuje upravovat zdroje použité v designéru oznámení

== Installation ==

Instalace pluginu

1. Nahrajte složku s pluginem (`odwp-notices_generator`) do adresáře `wp-content/plugins` vaší instalace aplikace [WordPress](https://wordpress.org)
2. Aktivujte plugin prostřednictvím stránky `Pluginy` v administraci
3. Nastavte plugin a použijte _shortcode_ `[notice_designer]` na libovolné stránce či příspěvku
